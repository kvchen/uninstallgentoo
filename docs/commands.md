Non-Character Keys
==================

The listing contains all usable non-character keyboard commands that will be accepted. Alternatives for longer names are provided, separated by commas (for example, typing either `backspace` or `bksp` into chat will work).

### Directional Keys

* up
* left
* down
* right
* home
* end
* pageup, pgup
* pagedown, pgdn

### Command Keys

* escape, esc
* backspace, bksp
* space
* tab
* enter
* insert, ins
* delete, del

### Modifier Keys

* capslock, caps
* lshift
* rshift
* lgui
* rgui
* control, ctrl
* lctrl
* rctrl
* alt
* lalt
* ralt

### Function Keys

* F1
* F2
* F3
* F4
* F5
* F6
* F7
* F8
* F9
* F10
* F11
* F12
